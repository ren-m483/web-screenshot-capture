# 販売ページ用ドラフト（下書き）

## キャッチコピー案

App Storeの公開データから、次に作るべきアプリ案とMVP要件、開発プロンプトまで。

## 説明文（使ってよい表現）

- App Storeの公開データをもとに市場傾向を分析します
- ランキング・評価・レビュー傾向からアプリ企画の仮説を作成します
- 個人開発で狙いやすいアプリ案を提案します
- Claude Code / Codex向けの要件定義プロンプトを生成します
- すべて「取得可能な公開情報に基づく分析」です

## 避ける表現（使わないこと）

- App Storeの全レビューを完全取得
- 売上を正確に推定
- ダウンロード数を取得
- 必ず売れるアプリ案を提案

## 免責事項テンプレート

本ツールは、Appleが公開するRSS/iTunes Search・Lookup APIから取得可能な情報をもとに分析結果を
表示するものであり、売上・ダウンロード数を保証するものではありません。分析結果・スコア・
アプリ案は参考情報であり、成功や収益を保証するものではありません。
