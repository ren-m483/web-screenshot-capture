# トラブルシューティング

## App Store URLからアプリIDを取得できませんでした

URLに `/id1234567890` の形式が含まれているか確認してください。例:

```
https://apps.apple.com/jp/app/example-app/id1234567890
```

## ランキング取得が失敗する（403 / 502）

- Apple RSSやiTunes Lookup APIへの外部ネットワークアクセスが必要です。
  社内ネットワークやプロキシ環境でブロックされていないか確認してください。
- 短時間に連続で取得すると一時的にレート制限にかかることがあります。時間を空けて再試行してください。

## レビュー本文が取得できない

App Store公開レビューフィードは取得できる範囲が限定的で、常に本文が返るとは限りません。
評価点・評価件数・説明文をもとにした簡易分析、またはレビューCSVインポートをご利用ください。

## AI分析が「簡易分析」表示のまま詳細化されない

`/settings` でOpenAI / Claude / Gemini のいずれかのAPIキーを設定してください。
未設定の場合はルールベースの簡易分析にフォールバックする仕様です。

## APIキーを設定したのに反映されない

設定画面から保存したキーはDBに暗号化保存され、`.env`の値より優先されます。
保存後にページを再読み込みし、`/api/settings` のレスポンスで `openaiApiKeyConfigured` 等が
`true` になっているか確認してください。

## SQLiteのロックエラーが出る

複数プロセスから同時に `npm run dev` を起動していないか確認してください。
開発中に発生した場合は、一度サーバーを停止してから再起動してください。
